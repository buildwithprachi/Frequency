# Frequency

Frequency is a web application that allows users to search and discover songs and artists. The application integrates with a public music API to fetch real-time data and provides features like searching, filtering, and sorting to enhance user experience.

🎯 The goal of this project is to demonstrate the use of:

- API integration using fetch
- JavaScript array higher-order functions (filter, sort, map)
- Dynamic UI rendering
- Clean and responsive user interface design

🔗 API Used
- Last.fm API is used to fetch music data such as songs, artists, and related information.

✨ Features
- 🔍 Search for songs and artists
- 🎼 Filter results based on criteria (e.g., artist, genre)
- 📊 Sort results (e.g., popularity, name)
- 🖼️ Display music data with images and details
- ⚡ Dynamic and responsive UI

🛠️ Technologies Used

- JavaScript
- HTML5
- CSS
- Fetch API

⚙️ Setup and Installation

Clone the repository:
git clone https://github.com/your-username/music-explorer.git

Navigate to the project folder:
cd Frequency

Install dependencies:
npm install

Start the development server:
npm start

Open in browser:

http://localhost:3000
